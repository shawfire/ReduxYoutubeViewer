####Getting started with this repo#####
Checkout this repo, install dependencies, then start the gulp process with the following:

```
	> git clone https://github.com/shawfire/ReduxYoutubeViewer.git
	> cd ReduxYoutubeViewer
	> npm install
	> npm start
```
